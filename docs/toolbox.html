Name sys_proc_exit
Address 0xffe000
Purpose This function ends the execution of the current program and returns control to the Toolbox. The program can return a result code to the Toolbox, although this is not currently used.
Prototype void sys_proc_exit(short result)
Parameters
  • result : the result code that should be returned to the caller
C sys_proc_exit(0) // Quit the program with a result code of 0
Assembly 
lda #0              ; Set the result code to 0
jsl sys_proc_exit   ; Quit the program
